# 调用大模型API

##  学习目标
1. 模型调用：base-url & model-name & 模型调用的基本组件 & 模型调用的基本流程/方式（流式/非流式）& 模型调用的请求写法 & 非流式响应接收
2. 工程：环境配置 & 环境加载（env）

## 练习任务
1. 会用 env 做环境变量的灵活管理（基本读写）
2. 会用 api key & api base url 进行大模型调用

## 练习思路

1. 环境变量配置读写
- os.Getenv() —— 读取当前环境变量的值，做兜底...
- 在开发环境可以用.env做临时文件，利用godotenv进行 load env文件（但是不会覆盖环境变量中已有的key）；在线上环境可以通过环境变量的手动注入：export xxx=xxx

2. 大模型调用

- 构造 questions - 用户问题：“任意输入”
- 构造 req - api 请求 = header（Authorization[Bearer apikey] + Content-Type[application/json]） + body（参数：模型名）+ 请求方法（post）+ 请求路径（baseURL+"/chat/completions"）
- 接收响应 resp - 输出ai回答

至于一些其他的东西，应该和 LLM厂商 相关。
